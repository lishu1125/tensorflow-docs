# 学习和使用机器学习

发布者注：*Deep Learning with Python* 介绍了 Python 语言和强大的 Keras 库在深度学习领域的应用。该书由 Keras 作者和 Google AI 研究员 FrançoisChollet 撰写，通过直观的解释和实际例子帮助你理解。
